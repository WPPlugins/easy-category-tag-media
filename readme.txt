Plugin let you assign categories or tags to your images or uploaded media.

== Description ==
* Plusin Help you assign categories or tags to your images or uploaded media
* Easy to filter media file by category or tag.

== Installation ==

1. Upload folder `easy-category-tag-media` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
